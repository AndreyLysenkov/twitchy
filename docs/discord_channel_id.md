# как найти id канала дискорд

## 1 способ

### написать в своей конфе любом канале в поле ввода `#имяКанала`;

![discord text field with #share in it](./img/discord_channel_id.md_1_1.png)

### поставить `\` перед `#`;

![discord text field with \#share in it](./img/discord_channel_id.md_1_2.png)

### отправить;

![discord sent text message with <#306906770700369926> in it](./img/discord_channel_id.md_1_3.png)

### скопировать циферки - они и есть искомый id;

![discord sent text message with <#306906770700369926> in it with highlighted 306906770700369926](./img/discord_channel_id.md_1_4.png)

## 2 способ

### открыть настройки клиента дискорд;

![discord interface with username, microphine, headsets and highlited User Settings buttons](./img/discord_channel_id.md_2_1.png)

### перейди в Appearence/Внешний вид -> Advanced/Расширенные и включить Developer Mod/Режим разработчика;

![discord user settings, with selected Appearence and turned on Developer Mod in Advanced](./img/discord_channel_id.md_2_2_rus.png)

### щелкнуть правой кнопкой мыши по желемому каналу и выбрать Copy Id/Копировать Id;

![discord share channel with seleceted context menu on it and highlighted Copy Id on that menu](./img/discord_channel_id.md_2_3.png)

### вставить id из буфера обмена;

![discord channel text field with ./add/allan_walpy/306906770700369926 in it and highlighted 306906770700369926](./img/discord_channel_id.md_2_4.png)
